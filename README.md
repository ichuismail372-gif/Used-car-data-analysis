👉 Download Dataset

📊 About Dataset
The dataset contains information about used cars collected from CarDekho. It can be used for:

- Data analysis (EDA)

- Price prediction

- Machine learning models (Kaggle)

📁 Features (Columns)
Main columns in the dataset:

- name – Car model name

- year – Manufacturing year

- selling_price – Car price

- km_driven – Kilometers driven

- fuel – Fuel type

- seller_type – Seller category

- transmission – Manual / Automatic

- owner – Number of previous owners (Kaggle)

🎯 Project Objective
- Understand car price trends

- Analyze factors affecting price (year, km driven, fuel, etc.)

- Build a simple prediction model

🛠️ Tools Used
- Python

- Pandas

- NumPy

- Matplotlib / Seaborn

- Jupyter Notebook

▶️ How to Use

📌 Conclusion
This project helps in understanding how different features affect car prices and can be used as a beginner-level machine learning project.
